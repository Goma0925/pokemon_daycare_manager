This directory houses application files.
These are html and php files, so front-end
and server side code to interact with the database,
which can be referenced inside of the sibling folder 'db'.

----------------------------------------------------------

In this application , we take on the MVC structure for 
ease of maintainability, modularity, and scaling. 

Relevant resources for the MVC model:

https://www.sitepoint.com/the-mvc-pattern-and-php-1/ -- conceptual and also explains idea of implementation
https://youtu.be/Fg0CP-ri87U -- video demonstrating how one can get started
https://softwareengineering.stackexchange.com/questions/139438/models-per-database-table -- addresses misconception and refines concept
